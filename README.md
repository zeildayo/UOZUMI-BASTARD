# 🙃 UOZUMI BASTARD (Development Build)
UOZUMI BASTARD は、特定の学習サイトにおける演習効率を最大化（あるいはハック）するためのユーザースクリプトです。
現在開発途上（v1.2）であり、サイトのアップデートによって動作が不安定になる可能性があります。

[!WARNING]
使用上の注意
本スクリプトは技術的興味および研究目的で作成されています。修了証の偽装機能などを含みますが、これによって生じた不利益や問題について、開発者は一切の責任を負いません。**「自己責任」**の意味がわかる方のみご利用ください。

🛠 主な機能
正解ハイライト (Answer Highlighting)

問題読み込み時の通信をフックし、questions.yaml から正解データを自動抽出。

選択肢ボタンに正解ラベルを表示し、背景色を変更します。

テキスト入力補助

記述式問題において、解析した正解をワンクリックで入力するボタンを生成します。

修了証の偽装 (Certificate Spoofing)

Canvas描画プロトコルを横取りし、発行名、IPアドレス、端末情報、発行時刻などを自由自在に書き換えます。

GUIコントロールパネル

画面左下にドラッグ可能な操作パネルを表示。リアルタイムで設定を変更可能です。

🚧 現在の状態：不安定 (Unstable)
このプロジェクトは現在進行形で開発されています。以下の点に注意してください：

DOM構造の変化に弱い: サイト側のアップデートでセレクタが変わると、ハイライト機能が停止する場合があります。

エラーハンドリング: 通信状況やブラウザの環境によっては、スクリプトが正しく注入されないことがあります。

未実装のバリデーション: 入力値のチェックが甘いため、無効な値を入力すると描画が崩れる可能性があります。

🚀 導入方法
ブラウザに Tampermonkey などのユーザースクリプトマネージャーをインストールします。

UOZUMI BASTARD.user.js の中身をコピーし、新規スクリプトとして保存してください。

対象サイト（https://joho.jun3010.me/*）にアクセスすると自動的にGUIが立ち上がります。
